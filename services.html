Services Provided by GCP

Google Cloud Platform offers a wide range of services to meet your computing, storage, and application needs.

Key Services Overview

1. Compute Engine

Infrastructure as a Service (IaaS) offering providing virtual machines for running applications.

2. App Engine

Platform as a Service (PaaS) offering for building and hosting web applications and services.

3. Kubernetes Engine

Managed Kubernetes service for container orchestration.

4. Cloud Storage

Scalable object storage for storing and retrieving data.

5. Cloud SQL

Fully managed relational database service.

6. BigQuery

Fully managed, serverless data warehouse for analytics and business intelligence.

7. Cloud AI

Suite of machine learning and artificial intelligence services, including APIs for vision, speech, etc.

8. Cloud Functions

Serverless compute service for running event-driven functions.

9. Cloud Networking

Services for building and managing networks, including VPC and Cloud Load Balancing.

Google Compute Engine

Delivers virtual machines running in Google's innovative data centers and worldwide fiber network.

Key Features

  • Custom VM shapes, persistent disks, local SSDs.
  • Global network infrastructure for high performance.
  • Automatic and committed use discounts.

Use Cases

  • Large-scale workloads and batch processing.
  • Containerized applications (GKE).

Google Kubernetes Engine (GKE)

Managed environment for deploying containerized applications. Run Kubernetes clusters without managing infrastructure.

Key Features

  • Automated provisioning and scaling.
  • Hybrid and multi-cloud capabilities.

Use Cases

  • Production container apps usually CI/CD pipelines.
  • Microservices architectures.

Google Cloud Storage

Scalable, durable object storage for data of all sizes.

  • Multiple storage classes (Standard, Nearline, Coldline, Archive).
  • Encryption at rest and in transit.
  • Ideal for backups, media streaming, and static websites.

Google Cloud SQL

Fully managed relational database (MySQL, PostgreSQL, SQL Server).

  • Automated backups and failover.
  • Built for high availability and scalability.

Google BigQuery

Serverless, highly scalable data warehouse for real-time analytics.

  • Analyze petabytes of data using standard SQL.
  • Built-in machine learning capabilities.

Google Vertex AI

Unified platform for building, training, and deploying ML models.

  • AutoML capabilities.
  • Support for TensorFlow, PyTorch, and more.

Google Cloud VPC & Load Balancing

Create secure virtual networks and distribute traffic globally.

  • Global load balancing (HTTP, TCP, UDP).
  • Customizable network topology and firewall rules.
